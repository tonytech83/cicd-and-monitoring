CI/CD & Monitoring - January 2026

<br/>

The “CI/CD & Monitoring” course provides a comprehensive overview of building, deploying, and observing CI/CD pipelines to automate software development and deployment processes. Participants will learn how to integrate version control systems, automate application builds and testing, and implement monitoring tools that ensure the stability and performance of both infrastructure and applications. The course is entirely hands-on, featuring numerous practical exercises that reinforce the acquired knowledge and prepare participants for real-world projects.
